For this project we will be using Housing Data to do exploratory data analysis and to clean out bad data. We will also be making new data columns and updating columns to make them more consistent.

Software Used:
  • SQL Server Management Studio (SSMS)
  • SQL Server
Standardizing Data Format:
  • Dates (2015-08-14)
  • Ensuring column data are consistent, for example, changing "Y" to "Yes" and "N" to "No"
Addressing Missing Data:
  • Attempt to input addresses for rows with missing data.
  • Utilize the ISNULL() function to determine null values.
  • Count the number of null values and take steps to update or remove them. In our case, we successfully updated null address values using a self-join.
  • Break the address into individual columns, including address, city, and state.
Key Actions Taken:
  • Extrapolate missing data by identifying similar parcel codes and addresses.
  • Conduct exploratory data analysis.
  • Fill missing data by using column averages.
  • Create common table expression (CTE) for analysis.
